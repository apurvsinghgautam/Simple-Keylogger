# Simple-Keylogger

This is a Simple Keylogger programmed in Python. It is used to record key strokes on the system on which it is executed.

## Prerequisites

- Python
- pythoncomm Library
- pyHook Library

## Installation

- `pip install pythoncomm`
- `pip install pyHook`

## Usage

- Open CMD
- Run `keylogger.py`
